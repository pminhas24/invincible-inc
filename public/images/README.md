## How to Swap In a Photo

1. Download the image and save it to this `images/` folder (e.g., `hero-bg.jpg`)
2. Open the relevant HTML file
3. Find the `<!-- REPLACE WITH REAL PHOTO: ... -->` comment
4. Replace the CSS background gradient with a CSS `background-image` rule, e.g.:

```css
/* Before (gradient placeholder): */
background: linear-gradient(135deg, #0a0a0f 0%, ...);

/* After (real photo with dark overlay): */
background: url('../images/hero-bg.jpg') center/cover no-repeat;
```

5. Keep the overlay `::before` pseudo-element in place to maintain text legibility over the photo.
