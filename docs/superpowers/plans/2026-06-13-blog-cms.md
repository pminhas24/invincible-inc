# Blog / Articles CMS Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a statically-generated `/blog` to the existing static HTML site, with content authored in Sanity and rebuilt automatically on publish via a Netlify webhook.

**Architecture:** Sanity (headless CMS, free) holds posts/categories/authors. Astro runs at the repo root and statically generates `/blog` and `/blog/[slug]` at build time using build-time GROQ queries (no client-side CMS calls — content is baked into HTML for SEO). The existing 30+ hand-coded pages are served as Astro `public/` passthrough, untouched. Netlify runs `astro build`; a Sanity webhook pings a Netlify build hook so publishing rebuilds the site.

**Tech Stack:** Astro (static), Sanity (`@sanity/client`, `@sanity/image-url`, `astro-portabletext`), Vitest (unit tests for pure helpers), Netlify (build + host + build hook).

**Source spec:** `docs/superpowers/specs/2026-06-13-blog-cms-design.md`

**Canonical domain (authoritative):** `https://invinciblesecuritygroup.com` (non-www) — matches existing `robots.txt`. Confirm Netlify primary domain in Phase 6; if it is `www`, swap everywhere.

- [ ] **Step 4: Implement the helper**

Create `src/lib/readingTime.ts`:

```ts
type Span = { _type: string; text?: string };
type Block = { _type: string; children?: Span[] };

const WORDS_PER_MINUTE = 200;

export function readingTimeMinutes(body: Block[]): number {
  const words = (body ?? [])
    .filter((b) => b._type === 'block')
    .flatMap((b) => b.children ?? [])
    .map((c) => c.text ?? '')
    .join(' ')
    .split(/\s+/)
    .filter(Boolean).length;
  return Math.max(1, Math.ceil(words / WORDS_PER_MINUTE));
}
```

- [ ] **Step 1: Create the sitemap endpoint**

Create `src/pages/blog-sitemap.xml.ts`:

```ts
import type { APIRoute } from 'astro';
import { sanity } from '../lib/sanity';

const SITE = 'https://invinciblesecuritygroup.com';

export const GET: APIRoute = async () => {
  const posts: { slug: string; _updatedAt: string }[] = await sanity.fetch(
    `*[_type == "post" && defined(slug.current)]{ "slug": slug.current, _updatedAt }`
  );
  const urls = [
    `<url><loc>${SITE}/blog</loc></url>`,
    ...posts.map(
      (p) => `<url><loc>${SITE}/blog/${p.slug}</loc><lastmod>${new Date(p._updatedAt).toISOString()}</lastmod></url>`
    ),
  ].join('');
  const xml = `<?xml version="1.0" encoding="UTF-8"?><urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">${urls}</urlset>`;
  return new Response(xml, { headers: { 'Content-Type': 'application/xml' } });
};
```
